O que é a
MP Soluções

A MP Soluções Digitais é minha empresa de freelance — mas também é o projeto mais técnico que já construí. Além de um site institucional, desenvolvi um pipeline completo de geração de sites com IA que automatiza do onboarding do cliente até o deploy final.

O diferencial: sou o único freelancer em Brasília que usa Claude Code com skills e agentes proprietários para entregar sites premium. O que levaria dias de trabalho manual é feito em horas, com qualidade consistente e checklist automatizado.

↳ O pipeline

Do brief
ao deploy

Cada projeto de site passa pelas mesmas etapas automatizadas, garantindo qualidade e velocidade de entrega.

01
/intake-cliente
Coleta automática
de dados

CNPJ → ReceitaWS busca nome, email, telefone, endereço e CNAE. CEP → ViaCEP completa o endereço. Google Maps URL → extrai GPS. Site existente → lê cores, fontes, WA e textos. Tudo automático antes de perguntar qualquer coisa ao cliente.

02
[CONFIRMAR]
Validação
com o cliente

Brief gerado é apresentado ao cliente para confirmar diferenciais, horários e informações que a IA não consegue buscar. Ponto de supervisão humana no processo.

03
/orcamento-cliente
Proposta
em HTML

Com o brief confirmado, gera proposta comercial em HTML responsivo com tier de preço, escopo, prazo e forma de pagamento. Pronto para enviar por email ou WhatsApp.

04
Pipeline CLAUDE.md
Geração
do site

Após aprovação, o CLAUDE.md orquestra a geração completa do site: HTML/CSS/JS modulares, banco de fotos Unsplash por segmento, efeitos 3D/GSAP, Schema.org, Open Graph, WhatsApp flutuante e CHECKLIST_MANUAL.md.

05
/relatorio-cliente
Entrega
documentada

Documento de entrega em HTML com screenshots, URL do site, tecnologias usadas, orientações de uso e próximos passos. Profissional e reutilizável.

06
/atualizar-cliente
Manutenção
cirúrgica

Para edições futuras: skill que recebe a pasta do cliente e o que precisa mudar, fazendo edições precisas sem quebrar o restante do site.

↳ Skills instaladas

As 7
skills proprietárias

Todas disponíveis globalmente no Claude Code — funcionam em qualquer pasta de cliente.

/intake-cliente
Busca automática de dados via CNPJ, CEP, Maps, site e redes sociais. Gera brief estruturado pronto para o pipeline.
/orcamento-cliente
Proposta comercial em HTML com tier de preço, escopo e prazo. Pronta para envio por email ou WhatsApp.
/atualizar-cliente
Edições cirúrgicas em sites entregues. Recebe pasta + o que mudar. Não quebra o restante do código.
/relatorio-cliente
Documento de entrega profissional com screenshots, URL, stack usada e orientações para o cliente.
/efeitos-3d
Biblioteca de efeitos prontos: CSS 3D, Three.js e GSAP ScrollTrigger. Código completo para colar no projeto.
/banco-de-fotos
IDs curados do Unsplash por segmento de negócio. Fotos estáveis, sem API key, licença comercial gratuita.
